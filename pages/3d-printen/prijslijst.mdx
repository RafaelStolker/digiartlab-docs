# Prijslijst

Voor 3D printen hanteren we een prijs per meter gebruikt filament. De prijzen zijn gebaseerd op onze werkelijke materiaalkosten met een toeslag van ‚Ç¨ 0,1000 per meter. Daarmee dekken we het onderhoud van de 3D printers.

Het gebruikte filament wordt automatisch uitgerekend door de slider.

Voor het voorbeeld gebruiken we een print waar 5 meter filament voor nodig is. Daar kan je grofweg 4cm x 4cm x 2cm van printen. De exacte hoeveelheid filament is afhankelijk van je ontwerp en slicerinstellingen.

| Materiaal | Prijs per meter (‚Ç¨) | Voorbeeld (5 meter) |
| --------- | ------------------- | ------------------- |
| PLA       | ‚Ç¨ 0,1894            | ‚Ç¨ 0,95              |

<Callout type="warning">
  **Let op!** Eigen materialen moeten eerst worden goedgekeurd door een

</Callout>
